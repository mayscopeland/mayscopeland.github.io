Projects

  • DraftKick (https://draftkick.com)

    Technology: JavaScript (Vue)

    Initial Release: 2022

    Powerful and easy-to-use fantasy draft software

    DraftKick is modern fantasy draft software. Install offline or use in your browser. It can handle all kinds of custom leagues using my proven z-scores methodology.

  • Projectile (https://projectile.pro)

    Technology: Python (Django)

    Initial Release: 2021

    Projectile = Open Projections + LastPlayerPicked Price Guide

    Projectile takes the continually updating Open Projections and uses them to compute fantasy values.

  • Open Projections (https://open-projections.com)

    Technology: Python (Flask)

    Initial Release: 2021

    The Open Projections are open source baseball player forecasts.

  • Price Guide 2.0 (https://github.com/mayscopeland/priceguide)

    Technology: Python

    Initial Release: 2021

    A rewrite of the original LastPlayerPicked Price Guide, using Python.

  • @SavesBot (https://twitter.com/SavesBot)

    Technology: PHP

    Initial Release: 2020

    @SavesBot is a simple Twitter bot that watches the FanGraphs Closer Depth Chart and tweets about changes.

  • Fantasy Wayback (https://fantasywayback.com)

    Technology: Static site built with PHP

    Initial Release: 2020

    Fantasy Wayback calculates fantasy values for historical baseball seasons, going back to the start of the fantasy era (1980).

    Valuations are made using the LastPlayerPicked Price Guide.

  • Price Guide User Scripts

    Technology: JavaScript

    Initial Release: 2009

    While the Price Guide is primarily a pre-season tool for draft prep, it can also generate in-season values. These simple JS scripts would insert in-season dollar values onto your Yahoo, CBS, or ESPN league pages.

  • LastPlayerPicked Price Guide

    Technology: PHP

    Initial Release: 2008

    My first public project was the LastPlayerPicked Price Guide, an online tool that created fantasy salaries for various projections, customized for any league configuration.

    I maintained this site for the next five years before publishing the source on GitHub.
